# Orbit Rescue Lab

NASA Space Place처럼 아이들이 먼저 놀아보고, 그다음 짧은 과학 규칙을 읽을 수 있는 웹 기반 우주 학습 게임 스타터입니다.

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다. 별도 설치가 필요 없습니다.

## 구조

```text
.
├── index.html
├── src
│   ├── main.js
│   └── styles.css
├── assets
│   └── .gitkeep
└── docs
    └── PROJECT_STRUCTURE.md
```

## 현재 게임

- 주제: 중력과 궤도
- 조작: 행성을 놓거나 드래그한 뒤 발사 버튼 누르기
- 목표: 탐사선을 위성까지 보내기
- 단계: 기본 궤도, 높은 위성, 소행성 피하기, 좁은 길 통과
- 추가 활동: `별 + 5 = 7`에서 양변에 `-5`를 놓아 `별 = 2`를 만드는 방정식 실험
