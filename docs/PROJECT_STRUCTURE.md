# Orbit Rescue Lab project structure

This starter is a dependency-free web app for a kid-friendly space learning game.

```text
.
├── index.html
├── src
│   ├── main.js
│   └── styles.css
├── assets
│   └── .gitkeep
└── docs
    └── PROJECT_STRUCTURE.md
```

## Page flow

1. Start with the playable experiment.
2. Keep the science rule short and concrete.
3. Add a hands-on equation lab for algebra reasoning.
4. Add related missions after the first experience.

## Next feature ideas

- Add sound feedback for launch, near miss, and success.
- Add a parent or teacher note section with vocabulary.
- Store completed missions in local storage.
- Move level data into a separate JSON file when the mission list grows.
